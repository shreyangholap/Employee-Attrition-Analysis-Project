# Employee Attrition Analysis — HR Analytics Dashboard

This project analyzes employee attrition factors using HR data and visualizes insights through an interactive Power BI dashboard. It helps organizations understand why employees leave and identify key drivers of attrition.


## Dashboard Overview

The dashboard includes:

- **Total Employees**
- **Total Attrition**
- **Attrition Rate**
- **Attrition by Department**
- **Attrition by Job Role**
- **Attrition by Age Group**
- **Attrition by Salary Band**
- **Attrition by Years at Company**
- **Overtime vs Attrition**

These insights support HR teams in making data-driven decisions to reduce employee turnover.


## Data Preparation

- Cleaned missing values  
- Converted data types  
- Created calculated columns: AgeGroup, SalaryBand  
- Prepared data for Power BI visualizations  


## Key Insights

- Employees aged **< 30** have the highest attrition  
- **Sales** and **Research** departments show highest turnover  
- Employees with **low salary bands** tend to leave more  
- **Overtime** strongly correlates with attrition  
- Employees with **< 3 years** experience are most likely to leave  


## Tools Used

- **Power BI**
- **Power Query**
- **DAX Measures**
- **HR Analytics Dataset (Kaggle)**
